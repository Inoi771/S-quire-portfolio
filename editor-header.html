⚙️ アプリ設定

Drive フォルダとマスタースプレッドシートの ID を設定します。未設定の場合は「自動作成」で新規作成できます。

音声ファイルをホストする GitHub リポジトリの情報を設定します。

音声ファイルを自動生成する際に使用する Google Cloud TTS の API キーです。

❓ 操作ガイド

✏️ エディタータブ — 基本操作フロー

  1. 1画面上部のドロップダウンで 年度 → 教科書 → 学年 → レッスン を順に選択してください。
  2. 2✏️ 編集 ボタンをクリックすると、左パネルに単語・英文リストが表示されます。
  3. 3左パネルの 単語（A タブ） または 英文（B タブ） を選択します。
  4. 4リスト内の単語・英文を右側の グリッド（マス目） にドラッグ&ドロップして配置します。
  5. 5配置が完了したら 💾 保存 ボタンをクリックします。
💡 新しいレッスンを作成するには、レッスン選択欄で 「新規レッスン名を入力」 を選び、レッスン名を入力して「編集」をクリックしてください。

🖱️ グリッドの操作

  1. 1セルをダブルクリック すると、英語・発音・日本語を直接編集できます。
  2. 2セルを右クリック（またはセル内の削除ボタン）で、そのセルの単語を削除できます。
  3. 3グリッド内でセル同士をドラッグすると、位置を入れ替えられます。
⚠️ 変更後は必ず 💾 保存 をクリックしてください。保存前に別のレッスンを開くと変更が失われます。

📝 新しい単語・英文の登録

  1. 1エディターを開いた状態で、左パネル下部の ＋ 単語を登録 または ＋ 英文を登録 ボタンをクリックします。
  2. 2英語・発音・日本語を入力して登録すると、マスターデータに追加されます。
  3. 3登録した単語・英文は左パネルに即座に表示されます。グリッドにドラッグして使用してください。
💡 登録時に Google Cloud TTS 設定済みの場合、音声ファイルが自動生成されます。
⚠️ 音声生成に失敗した場合は、画面右下に通知が表示されます。「🔄 再生成する」で再試行できます。

📚 単語帳作成タブ — PDF の生成

  1. 1上部の 📚 単語帳作成 タブをクリックします。
  2. 2年度・教科書・学年を選択して 📥 読込 ボタンをクリックします。
  3. 3左パネルから PDF に含めるレッスンを右パネルにドラッグして並べます。
  4. 4📄 PDF を生成 ボタンをクリックすると、Google Drive の「生成PDF」フォルダに保存されます。

🔊 音声ファイルの自動生成（TTS）

  1. 1単語または英文を登録・編集すると、Google Cloud TTS で音声が自動生成されます。
  2. 2音声生成に失敗した場合は、画面右下に 「⚠️ 音声生成に失敗しました」 の通知が表示されます。
  3. 3通知内の 「🔄 再生成する」 をクリックすると再試行します。
  4. 4登録自体を取り消す場合は 「❌ 登録をキャンセル」 をクリックします。編集の取り消しは 「↩️ 英語を元に戻す」 です。
💡 TTS を使用するには ⚙️ 設定 タブで GOOGLE_CLOUD_TTS_API_KEY と GITHUB_TOKEN、GITHUB_BASE_URL を設定してください。

⚙️ 設定タブ — 各項目の説明

  1. 1ENGLISHWORDS_FOLDER_ID：音声ファイルや年度フォルダを格納する Google Drive フォルダの ID。未設定の場合は「🔧 Drive リソースを自動作成」で生成できます。
  2. 2ENGLISHWORDS_SHEET_ID：単語・英文のマスターデータが入ったスプレッドシートの ID。自動作成で生成できます。
  3. 3GITHUB_BASE_URL：音声ファイルをホストする GitHub リポジトリの Raw コンテンツ URL（例：https://raw.githubusercontent.com/ユーザー名/リポジトリ名/main）。
  4. 4GITHUB_TOKEN：音声ファイルを GitHub にアップロードするための Personal Access Token（repo スコープ必要）。
  5. 5GOOGLE_CLOUD_TTS_API_KEY：音声自動生成に使う Google Cloud Text-to-Speech の API キー。
  6. 6HOMEPAGE_URL（任意）：ロゴをクリックしたときのリンク先 URL。
  7. 7STUDENT_HOMEPAGE_URL（任意）：生徒用アプリの URL。
💡 新しい年度を追加するには、GAS エディタで initializeYearResources('2027年度版') を実行してください。
