👥 スタッフのアクセス管理

スタッフを追加する手順

  1. 管理タブ → ユーザー管理 → 「アクセスを許可」でスタッフのメールアドレスを追加
  2. スタッフがアプリ（https://fir-quire.web.app）にGoogleアカウントでログイン
    → この時点でスタッフIDが自動発行されます
  3. LINE通知を使う場合：スタッフがLINEボットに自分のメールアドレスを送信

※ アクセスを取り消す場合は、管理タブ → ユーザー管理 → 「削除」をクリックしてください。Drive共有とFirebase認証の両方が自動で解除されます。

🔄 スタッフデータのリセット手順

スタッフの登録をすべてやり直したい場合（例：アーキテクチャ変更後のクリーンスタート）は以下の手順で行います。

⚠️ 注意： 以下の操作を行うとスタッフのLINE登録と通知設定がすべて消去されます。スタッフは再度LINE登録が必要になります。

  1. GASエディタ（script.google.com）を開く
  2. 左メニュー「プロジェクトの設定」→「スクリプトプロパティ」を開く
  3. TEACHER_ID_MAP の値を {} に変更
  4. LINE_USER_MAPPING の値を {} に変更
  5. Firebaseコンソール → Firestore → allowedUsers コレクションを開き、すべてのドキュメントを削除
  6. アプリに戻り、管理タブ → ユーザー管理 → 「Firebaseセキュリティ初期化」を実行して管理者を再登録
  7. 各スタッフに再ログインを依頼（ログイン時にIDが自動発行されます）

💬 LINE通知の設定手順（管理者向け）

スタッフにLINEで通知を届けるための初期設定です。Gmailだけでよい場合はこの設定は不要です。

Step 1：LINE Developers でチャンネル作成
  1. https://developers.line.biz/ にアクセス
  2. 「Messaging API」チャンネルを新規作成
  3. 「チャンネル設定 → チャンネルアクセストークン（長期）」を発行してコピー
Step 2：アプリにトークンを設定

管理タブ → 設定 → スクリプトプロパティ → 「LINE_CHANNEL_ACCESS_TOKEN」にStep 1でコピーしたトークンを貼り付けて保存

Step 3：Webhook URLを設定

LINE Official Account Manager → 「設定 → Messaging API → Webhook URL」に以下を入力：

https://script.google.com/macros/s/AKfycbzrzZkyS42v_-kNNrmR4NumrVxfjdwNeJ0uCk3k5mha88Dm7ZarVzjVAkDY8WIqpKybWw/exec

「Webhookの利用」をONにして保存

Step 4：スタッフがLINE登録

スタッフが LINE公式アカウントを友だち追加し、アプリに登録しているメールアドレスをメッセージで送信 → 「登録完了」と返信が届けば完了。自動的にLINE User IDが紐づきます

📚 講習管理・通常授業設定の使い方

📩 LINEスケジューラーの使い方

毎月の定型連絡（室長用・ミーティング・報告書）を自動生成してLINEまたはGmailで送信する機能です。

3種類の自動連絡
  • 室長用連絡：毎月の引落データ送信日・締切日・講習名を含む連絡。送信先は手動で選択
  • 全体ミーティング：ミーティング日程の前日に全スタッフへ自動通知
  • 回数報告書：提出日前日に全スタッフへ自動通知
使い方
  1. 管理タブ → LINE通知 を開く
  2. 年月を選ぶと3種類のスケジュールが自動生成されます
  3. 内容を確認・編集して「保存」
  4. 「今すぐ送信」で手動送信、または自動送信トリガーを有効にすると予定時刻に自動送信されます

⚠️ 自動送信するには「トリガーを有効にする」ボタンを押す必要があります。LINE通知タブの下部から設定できます。

🤖 AIナレッジベース（チャットボット）の使い方

🎨 チラシの作り方

💻 パソコンでの操作を推奨
チラシ作成はA4サイズのプレビューを見ながら作業するため、パソコンの大きな画面での操作をおすすめします。
スマートフォンでも基本操作はできますが、プレビューが小さく表示されるため細かい確認が難しくなります。
📋 基本の流れ
① 上部で校舎を選ぶ
② チャットでAIに指示を送る（例：「チラシを作って」）
③ プレビューを確認しながら修正
④ 完成したら「💾 保存」→「📥 PDF」
💬 AIチャットで指示する
右側のチャット欄にやりたいことを日本語で入力して「送信」をタップ。例：
・「夏期講習のチラシを作って」
・「キャッチコピーを変えて」
・「料金表を目立たせて」
・「全体的にもっとポップな感じにして」
何度でも修正指示を送れます。
📊 日程・料金データを入れる
「日程データを含める」「料金データを含める」にチェックを入れてからAIに指示すると、登録済みのデータが自動的にチラシに組み込まれます。
※先に講習管理タブで日程を作成しておく必要があります。
🖼️ 画像を入れる
「メイン画像」エリアで画像を選択するか、新しい画像を「📤 UP」でアップロードします。
画像を選んだ状態でAIに「画像を入れて」と指示すると、チラシに挿入されます。
✏️ 編集モード（PC向け）
「✏️ 編集モード」ボタンをONにすると、プレビュー上のテキストを直接クリックして書き換えられます。
・クリックした文字の上にツールバーが出ます
・A- / A+：文字サイズの変更
・B：太字にする
・🎨：文字の色を変える
※AIに指示を送ると、手動編集した内容が一部変わる場合があります。
📥 PDF出力
「📥 PDF」ボタンでA4サイズのPDFをダウンロードできます。
印刷用にトンボ（裁断の目印線）を付けたい場合は、PDFボタンの上にある「✂️ トンボ：あり」を選んでください。
💾 保存について
「💾 保存」ボタンを押すと、現在のチラシとチャット履歴が保存されます。校舎ごとに別々のチラシを保存できます。
「📋 共通」を選ぶと全校舎共通のチラシとして保存されます。
次回この画面を開いたとき、保存した状態から続きを編集できます。
🗑️ リセット
アクションボタン右端の「🗑️」ボタンで、チャット履歴とプレビューをすべてクリアして最初からやり直せます。

📄 写真・PDF から読み込む

写真またはPDFを選択すると、全生徒の成績を自動で読み取って保存します。
